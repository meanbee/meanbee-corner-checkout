    rem 

    @mixin rem($property, $values) { ... }
    View source

    Description

    Mixin to output px or rem value depending on application file for a given property.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $property

    Property name to output

    		 String — none
    $values

    Values to assign to the property

    		 Length — none

    Example

    Usage

    .foo {
  @include rem(font-size, 100px);
}

.bar {
  @include rem(font-size, 100px);
}

    CSS output within application.scss

    .foo {
  font-size: 6.25rem;
}

    CSS output within application-ie8.scss

    .bar {
  font-size: 100px;
}

    before-ie-10 

    @mixin before-ie-10() { ... }
    View source

    Description

    Mixin to output content if application file is for ie10 or less

    Parameters

    None.

    Example

    Usage

    .foo {
  background: url(../images/cat-icon.svg);
  @include before-ie-10 {
     background: url(../images/ie/cat-icon.png);
  }
}

    CSS output within application.scss

    .foo {
   background: url(../images/cat-icon.svg); }

    CSS output within application-ie9.scss

    .foo {
   background: url(../images/cat-icon.svg);
   background: url(../images/ie/cat-icon.png); }

    breakpoint 

    @mixin breakpoint($min, $max) { ... }
    View source

    Description

    Mixin to output breakpoint given a variable, sass map reference or value

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $min

    Value to assign to min-width within media query

    		 Length — none
    $max

    Value to assign to max-width within media query

    		 Length — none

    Example

    Usage

    .foo {
  @include breakpoint(100px) {
     width: 50px;
  }
}

.bar {
  @include breakpoint(200px, 400px) {
     width: 50px;
  }
}

    CSS output

     @media (min-width: 100px) {
   .foo {
      width: 50px; } }

 @media (min-width: 200px) and (max-width: 400px) {
   .bar {
      width: 50px; } }
